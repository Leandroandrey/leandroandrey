# Olá, eu sou o Leandro Andrey 👋

**Software Developer | Angular 19 | .NET MAUI | React (Vite) | Apaixonado por IA**

Sou formado em Análise e Desenvolvimento de Sistemas e atuo profissionalmente como Desenvolvedor de Software. Minha carreira é movida pela constante evolução tecnológica. Deixei o nível iniciante para trás e hoje foco em arquiteturas modernas, escaláveis e na criação de experiências de usuário excepcionais.

Atualmente, domino o ecossistema **Angular 19** no front-end profissional, exploro o desenvolvimento mobile/desktop cross-platform com **.NET MAUI (Latest)** e estou imerso no ecossistema moderno de **React + Vite**. Minha grande paixão do momento é a **Inteligência Artificial**, buscando integrar soluções inteligentes e automações no ciclo de desenvolvimento.

Para consolidar minha visão de ponta a ponta, estou cursando Pós-Graduação em Fullstack Development for Cloud Native Systems pela UNESC e aprimorando meu Inglês para desafios globais.
